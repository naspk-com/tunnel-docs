使用 Cloudflare Tunnel 需要两样东西：一个域名和一个 Cloudflare 账号。这篇文章先解决域名的问题——很多人第一次买域名，面对腾讯云、阿里云、GoDaddy、Namecheap 一堆平台，不知道选哪个，也不知道买完域名之后要做什么。本文聚焦国内用户最常用的腾讯云和阿里云，手把手教你注册域名。

## 域名基础知识

在注册之前，先了解几个关键概念：

### 域名结构

以 `nas.example.com` 为例：

- `com` 是**顶级域名**（TLD，Top-Level Domain）
- `example` 是**二级域名**（SLD，Second-Level Domain），也就是你注册的部分
- `nas` 是**子域名**（Subdomain），可以免费创建任意多个

**你花钱买的是二级域名**（如 `example.com`），子域名可以在 DNS 管理面板中免费创建。

### 常见顶级域名

| 后缀      | 含义 | 价格（参考）    | 适合场景     |
| ------- | -- | --------- | -------- |
| `.com`  | 商业 | ￥60-80/年  | 通用，首选    |
| `.cn`   | 中国 | ￥30-50/年  | 需备案，国内场景 |
| `.top`  | 顶级 | ￥10-20/年  | 价格低，个人项目 |
| `.xyz`  | 通用 | ￥10-15/年  | 价格低，测试用  |
| `.me`   | 个人 | ￥80-100/年 | 个人博客     |
| `.site` | 网站 | ￥20-30/年  | 个人项目     |

**建议：** 长期使用选 `.com`；只是测试或预算有限，`.top` 或 `.xyz` 也完全够用。

### 域名备案

- **使用 Cloudflare Tunnel 不需要备案**。因为流量走 Cloudflare 全球网络，不经过国内服务器。
- 如果你后续想在国内服务器上托管网站，`.cn`、`.com` 域名或使用国内服务器时才需要备案。
- 对于 NAS 内网穿透场景，建议选择一些价格低的域名后缀（如 `.top`、`.xyz` 等）。

## 平台对比：腾讯云 vs 阿里云

| 对比维度 | 腾讯云 | 阿里云 | 国外平台（如 Namecheap、GoDaddy） | Cloudflare Registrar |
|---------|--------|--------|-----------------------------------|---------------------|
| 域名价格 | 与阿里云接近 | 与腾讯云接近 | 通常略高，常有首年优惠码 | 部分域名首年比国内高，部分持平 |
| 首年优惠 | 经常有 1 元/首年活动 | 经常有 1 元/首年活动 | 新用户优惠码，约 $1-3 | 无首年优惠 |
| 续费价格 | 正常市场价 | 正常市场价 | 偏高，部分域名首年便宜续费很贵 | 续费普遍比国内高 |
| 实名认证 | 需要（工信部要求） | 需要（工信部要求） | 不需要 | 不需要 |
| 支付方式 | 微信/支付宝/网银 | 支付宝/微信/网银 | 信用卡/PayPal | 信用卡/PayPal |
| DNS 管理 | 免费 DNS 解析 | 免费 DNS 解析 | 免费 DNS 解析 | 自带 Cloudflare DNS |
| 客服支持 | 在线客服 + 工单 | 在线客服 + 工单 | 英文工单/在线客服 | 工单 + 社区 |
| 转出便利性 | 支持，需实名 | 支持，需实名 | 支持，需解锁 | 60 天锁定后自由转出 |

**结论：** 腾讯云和阿里云差别不大，哪个有账号就在哪个平台买。Cloudflare Registrar 虽然免实名且 DNS 托管一体化，但价格不便宜、续费普遍比国内高。**对于国内用户，建议首选腾讯云或阿里云**，价格有优势、支付方便、有中文客服。

## 方式一：在腾讯云注册域名

### 第一步：登录腾讯云

1. 访问 <https://cloud.tencent.com>。
2. 如果没有账号，点击「注册」，使用微信扫码或邮箱注册。
3. 注册完成后，完成**实名认证**（个人认证即可，需要身份证信息）。

![腾讯云控制台首页](/images/tencent-console.png)

### 第二步：搜索并购买域名

![腾讯云域名注册入口](/images/tencent-domain-entry.png)

1. **进入域名注册页面**
   - 在腾讯云控制台顶部搜索框，输入「域名注册」，点击进入域名注册服务页面。

![腾讯云域名搜索结果](/images/tencent-search-result.png)

2. **搜索域名**
   - 在搜索框中输入你想要的域名（如 `my-nas-home`），点击「查询」。
   - 系统会列出该域名在不同后缀下的可用情况和价格：
     - ✅ **可注册**：该域名可以购买。
     - ❌ **已注册**：已被别人占用，可尝试其他后缀。
     - 💰 **溢价域名**：价格较高，普通场景不建议选。

![腾讯云购物车页面](/images/tencent-cart.png)

3. **加入购物车**
   - 选择一个可用的后缀（建议 `.com`），点击「立即加购」。
   - 点击购物车下方的「立即购买」按钮，进入结算页。

![腾讯云结算页选择域名所有者](/images/tencent-owner-select.png)

4. **选择域名所有者**
   - 进入结算页后，需要选择「域名所有者」（即信息模板）。
   - 如果还没有创建信息模板，点击「新建模板」跳转创建（见第三步）。
   - 选择已审核通过的信息模板并勾选服务协议后，点击「提交订单」。

![腾讯云支付页面](/images/tencent-payment.png)

5. **确认信息并支付**
   - 确认订单信息无误后，点击「去支付」。
   - 跳转到支付页面，选择支付方式（微信/网银），完成支付。

### 第三步：域名信息模板（实名）

根据中国工信部规定，域名注册必须实名。**建议先创建模板，再购买域名，这样结算时可以直接选择。**

![腾讯云信息模板入口](/images/tencent-template-nav.png)

1. **进入信息模板页面**
   - 在域名注册页面，点击「立即创建」进入管理页面。

![腾讯云信息模板创建表单](/images/tencent-template-form.png)

2. **创建模板**
   - 建议勾选「使用账号信息填写」，这样可以快速填写基本信息。
   - 填写真实姓名、身份证号码、邮箱、手机号等个人信息。
   - **注意：** 所有信息必须与身份证完全一致，否则审核会失败。

![腾讯云身份证上传区域](/images/tencent-id-upload.png)

3. **上传身份证并提交**
   - 上传身份证正反面照片（支持 JPG/PNG）。
   - 确认信息无误后，点击「创建」。

![腾讯云信息模板状态](/images/tencent-template-status.png)

4. **等待审核**
   - 提交后模板状态为「审核中」，通常 1-2 个工作日完成。
   - 审核通过后状态变为「已实名认证」。
   - 审核期间不影响域名购买，但域名需要审核通过后才能正常使用。

> **小技巧：** 建议先创建信息模板并提交审核，审核期间可以去搜索和挑选域名，等模板审核通过后再直接结算购买，一步到位。

## 域名选购建议

### 起名技巧

- **简短好记**：3-10 个字符最佳，方便输入。
- **避免混淆字符**：数字 `0` 和字母 `O`、数字 `1` 和字母 `l` 容易混淆。
- **避免连字符**：`my-nas.com` 不如 `mynas.com` 好记，口述时也不方便。
- **与用途相关**：`myhome-nas.com`、`dustinks-nas.com` 等，一看就知道用途。

### 省钱技巧

- **首年优惠**：腾讯云和阿里云经常有 1 元/首年的新用户活动。
- **续费价格**：注意看续费价格，有些后缀首年便宜但续费很贵。
- **批量购买**：如果需要多个域名，可以关注活动套餐。
- **`.top` 和 `.xyz`**：这两个后缀常年低价，适合预算有限的用户。

## 常见问题

### 1. 域名注册后可以退款吗？

一般不支持退款。域名注册是即时生效的服务，注册成功后即占用该域名资源。

### 2. 域名注册后多久能生效？

支付成功后，域名会立即注册。但实名审核通常需要 1-2 个工作日。审核通过后，域名就正式属于你了。

### 3. 域名到期了怎么办？

域名到期后通常有 30 天左右的「续费宽限期」和 30 天的「高价赎回期」。建议在到期前及时续费，避免域名被释放。

### 4. 可以不实名吗？

国内注册商（腾讯云、阿里云）要求必须实名。如果不想实名，可以考虑海外注册商（如 Namecheap、Cloudflare Registrar），但支付方式可能不够方便。

### 5. 买完域名后要做什么？

下一步是将域名的 DNS 托管到 Cloudflare，这样才能使用 Cloudflare Tunnel 功能。详见 [域名接入 Cloudflare 管理](/prerequisites/domain-dns-setup)。
