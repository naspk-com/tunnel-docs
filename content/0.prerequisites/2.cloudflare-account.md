域名已经买好了，下一步就是注册 Cloudflare 账号。很多人第一步就卡住了——没有 Cloudflare 账号，也不知道该怎么注册。

其实注册 Cloudflare 账号完全免费，不需要绑信用卡，也不需要任何技术背景。这篇文章用截图级详细步骤，带你从零开始注册一个 Cloudflare 账号。

## 解决方案概览

Cloudflare 是全球最大的 CDN 和网络安全服务商之一，提供免费套餐，包含 CDN、DNS 管理、DDoS 防护、Tunnel 等核心功能。注册一个账号是使用 Cloudflare Tunnel 的前提条件。

![Cloudflare 官网首页](/images/cloudflare-homepage.png)

## 第一步：打开 Cloudflare 官网

1. 打开浏览器，访问 [https://www.cloudflare-cn.com/](https://www.cloudflare-cn.com/)。
2. 页面右上角点击「登录」按钮。
3. 你会进入登录页面，点击「注册」按钮。

![Cloudflare 登录页面](/images/cloudflare-login.png)

**注意事项：**
- 建议使用常用邮箱（Gmail、QQ 邮箱、163 邮箱等均可）。
- 如果页面显示为英文，可以在页面底部切换语言为「简体中文」。

## 第二步：填写注册信息

在注册页面中，你需要填写以下信息：

1. **邮箱地址**：输入你的邮箱，这将是你的 Cloudflare 账号。
2. **密码**：设置一个强密码（建议至少 12 位，包含大小写字母、数字和特殊字符）。
3. 点击「注册」按钮。

![Cloudflare 注册页面](/images/cloudflare-signup.png)

**密码安全建议：**
- 不要使用与其他网站相同的密码。
- 建议使用密码管理器（如 1Password、Bitwarden）生成和保存密码。
- Cloudflare 账号涉及域名管理权限，安全等级要足够高。

## 第三步：首次登录与基础设置

注册成功后，你会进入 Cloudflare 控制台。首次登录时，Cloudflare 可能会引导你完成一些基础设置。

### 添加站点（可跳过）

Cloudflare 会提示你「添加站点」（输入域名）。如果你还没有域名，可以先跳过这一步：

1. 在添加站点页面，你可以直接点击左侧导航栏的「Websites」跳过。
2. 或者，在页面底部找到「Skip」或「稍后再说」链接。
3. 即使不添加站点，你的账号也已经创建成功，可以使用 Account ID 和 API Token 了。

### 了解控制台界面

首次进入控制台，你会看到：

- **左侧导航栏**：包含帐户主页、最近访问、域名、计算、AI、Zero Trust 等菜单。
- **顶部账户信息**：显示你的账户名称和邮箱。
- **右上角头像菜单**：点击可以进入配置文件页面。

![Cloudflare 控制台首页](/images/cloudflare-dashboard.png)

## 第四步：准备后续配置所需的信息

注册完成后，有两个信息在后续使用 Cloudflare Tunnel 时会用到：

### 1. Account ID

Account ID 是你的 Cloudflare 账户的唯一标识，在配置 Tunnel 应用时需要填写。

**获取方式：**
- 登录 Cloudflare 控制台后，找到搜索框，输入「id」。
- 在搜索结果中，你会看到「复制帐户 ID」按钮。
- 点击复制按钮即可复制 Account ID。

![Cloudflare Account ID](/images/cloudflare-account-id.png)

### 2. API Token 的概念

API Token 是用于程序化访问 Cloudflare API 的凭证。在后续配置 Tunnel 应用时，你需要创建一个具有适当权限的 API Token。

**暂时不需要操作**——详见 [API 凭证](/advanced/api-credentials)。

## 常见问题

### 1. 注册 Cloudflare 账号需要付费吗？

不需要。Cloudflare 提供免费套餐（Free Plan），包含 DNS 管理、CDN、DDoS 防护、Tunnel 等核心功能。对于个人用户和 NAS 场景完全够用。

### 2. 需要绑定信用卡吗？

不需要。免费套餐不需要任何支付信息。只有升级到 Pro、Business 或 Enterprise 套餐时才需要绑定支付方式。

### 3. 一个邮箱可以注册多个账号吗？

不行，每个账号需要用不同的邮箱地址。Cloudflare 也支持通过 Google、Apple 等第三方账号快速登录。

### 4. 注册后可以改邮箱吗？

可以。登录后进入配置文件页面，在「设置」部分可以更新电子邮件。

### 5. 忘记密码怎么办？

在登录页面点击「Forgot password?」或「忘记密码？」，输入注册邮箱，Cloudflare 会发送密码重置邮件。
