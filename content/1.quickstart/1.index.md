Cloudflare Tunnel 是 fnOS 原生内网穿透应用，基于 Cloudflare 官方客户端适配。它让你无需公网 IP、无需配置路由器端口转发，就能将 NAS、Web 面板、Home Assistant 等内网服务安全地暴露到互联网。

## 功能模块

Cloudflare Tunnel 应用提供六个功能页面：

![应用首页仪表盘](/images/dashboard.png)

| 页面 | 说明 |
|------|------|
| **首页** | 查看 Tunnel 服务运行状态与运行时长，快速跳转到管理页面 |
| **系统配置** | 填写 Cloudflare 账号凭据，设置连接协议、IP 版本、开机自启 |
| **Tunnel 列表** | 创建/删除 Tunnel，连接/断开，配置域名转发规则 |
| **API 凭证** | 创建外部应用接入凭证，通过独立 HTTP API（端口 19092）和 MCP 服务（端口 19093）调用 Tunnel 接口 |
| **运行日志** | 查看 cloudflared 实时日志，支持关键字过滤与自动刷新 |
| **关于我们** | 查看项目信息与作者联系方式 |

## 三步上手

### 第一步：安装应用

在 fnOS 应用中心搜索「Cloudflare Tunnel」，一键安装。

详见 [安装指南](/quickstart/installation)

### 第二步：配置账号

在「系统配置」页面填写 Cloudflare 账号 ID 与 API 令牌，测试连接后保存。

详见 [Cloudflare 账号配置](/configuration/cloudflare-credentials)

### 第三步：创建并连接 Tunnel

在「Tunnel 列表」页面新建一个 Tunnel，点击「连接」，然后配置域名到内网服务的转发规则。

详见 [创建 Tunnel](/tunnel-management/manage-tunnels#创建-tunnel)
