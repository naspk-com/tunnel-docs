## 通过应用中心安装

Cloudflare Tunnel 作为 fnOS 原生应用，可通过 fnOS 应用中心一键安装，无需手动下载或配置依赖。

![fnOS 应用中心安装](/images/installation.png)

### 安装步骤

1. 打开 fnOS Web 管理后台
2. 进入「应用中心」
3. 搜索「Cloudflare Tunnel」或「Tunnel」
4. 点击「安装」，等待下载完成
5. 在安装向导中，填写 Cloudflare 账号凭据，**Account ID** 和 **API Token** 获取方式请看 [Cloudflare 账号配置](/configuration/cloudflare-credentials)
6. 点击「下一步」，等待应用安装完成

![fnOS 应用中心安装配置](/images/installation-config.png)

应用安装完成后，应用会出现在 fnOS 侧边栏中。点击即可进入 Tunnel 管理界面。

## 依赖说明

Cloudflare Tunnel 应用内置了 `cloudflared` 客户端，**无需额外安装任何依赖**。

应用会根据你的设备架构自动选择对应版本的 cloudflared：

| 架构 | 支持 |
|------|------|
| amd64（x86_64） | 支持 |
| arm64（aarch64） | 支持 |
| arm（armv7l） | 支持 |
| i386（i686） | 支持 |

## 首次启动

安装后首次打开应用，你会看到**首页仪表盘**，显示 Tunnel 服务处于「已停止」状态。

![首次启动首页](/images/dashboard-stopped.png)

此时需要先完成账号配置，才能开始使用 Tunnel 功能。点击「账户配置」按钮进入配置页面，或参见 [Cloudflare 账号配置](/configuration/cloudflare-credentials)。

## 更新应用

当 fnOS 应用中心提示有新版本时，点击更新即可。更新不会丢失已有的配置和 Tunnel 信息。

## 卸载应用

如需卸载，在 fnOS 应用中心找到 Cloudflare Tunnel，点击卸载。卸载前建议：

1. 在应用中断开所有 Tunnel 连接
2. 如有需要，先在 Cloudflare 控制台清理不再使用的 Tunnel 和 DNS 记录
