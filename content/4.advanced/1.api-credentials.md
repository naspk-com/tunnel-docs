Cloudflare Tunnel 应用提供两种方式让外部应用和 AI 智能体接入：

- **独立 HTTP API**（端口 19092）：通过凭证方式调用 Tunnel 管理接口
- **MCP 服务**（端口 19093）：将全部 Tunnel 管理能力暴露给 AI 智能体，支持 SSE 协议

API 凭证页面用于管理这两种接入方式所需的凭证。

> **注意事项**：API 和 MCP 功能仅在 1.0.7 及以上版本可用。

![API 凭证列表](/images/api-credentials-list.png)

## 应用场景

API 凭证适用于以下场景：

- 在自建面板或脚本中集成 Tunnel 管理功能
- 让其他 fnOS 应用调用 Tunnel 接口
- 自动化运维场景（如定时检测 Tunnel 状态）

## 创建凭证

1. 进入「API 凭证」页面
2. 点击「创建凭证」
3. 填写信息：
   - **应用名称**：标识调用方，便于后续管理（如 `com.dustinky.qwenpaw`）
   - **备注**：可选，记录用途说明
4. 点击创建

![创建 API 凭证](/images/api-credentials-create.png)

### 凭证信息

创建成功后，系统会显示生成的凭证信息：

- **appId**：应用唯一标识（如 `app_com.dustinky.qwenpaw_2f1c7598`）
- **appKey**：应用密钥，用于请求认证

![凭证创建成功](/images/api-credentials-created.png)

> **重要提醒**：`appKey` 仅显示一次，请立即复制并妥善保存。关闭此弹窗后无法再次查看 appKey。

## 管理凭证

### 查看凭证列表

凭证列表显示所有已创建的凭证，包含：

- 应用名称
- 创建时间
- 备注信息

### 删除凭证

点击凭证右侧的删除按钮即可移除。删除后该凭证立即失效，使用该凭证的外部应用将无法调用 API。

## 安全建议

- 只给信任的应用创建凭证
- 不要将 `appKey` 硬编码到前端代码中
- 如果凭证泄露，立即删除并重新创建
- 定期清理不再使用的凭证

## MCP 服务

Tunnel 应用内置了 MCP（Model Context Protocol）服务，运行在端口 19093，通过 SSE（Server-Sent Events）协议将全部 22 个 Tunnel 管理工具暴露给 AI 智能体。

### 连接信息

| 项目 | 值 |
|------|-----|
| 地址 | `http://<你的NAS IP>:19093/sse` |
| 传输方式 | SSE (Server-Sent Events) |
| 工具数量 | 22 个 |
| 工具分类 | 状态监控、Tunnel 管理、域名 DNS、配置管理、凭证管理 |

### 认证方式

MCP 服务支持两种认证方式：

**方式一：URL 参数（推荐）**

适用于所有 MCP 客户端（如 QwenPaw），将凭证信息通过 URL 参数传递：

```
http://<你的NAS IP>:19093/sse?appId=<你的AppID>&appKey=<你的AppKey>
```

**方式二：请求头**

适用于 QwenPaw、OpenClaw 等支持自定义 Header 的客户端：

```
X-App-Id: <你的AppID>
X-App-Key: <你的AppKey>
```

### QwenPaw 配置

编辑 `qwenpaw_config.json`：

```json
{
  "mcpServers": {
    "fnos-tunnel": {
      "type": "sse",
      "url": "http://<你的NAS IP>:19093/sse",
      "headers": {
        "X-App-Id": "<你的AppID>",
        "X-App-Key": "<你的AppKey>"
      }
    }
  }
}
```

### 可用工具

| 工具名 | 功能描述 |
|--------|----------|
| `tunnel_status` | 获取 Tunnel 运行状态 |
| `tunnel_logs` | 获取最近 200 条日志 |
| `tunnel_clear_logs` | 清空运行日志 |
| `tunnel_get_cloudflare_config` | 获取 Cloudflare 配置 |
| `tunnel_test_cloudflare_config` | 测试 Cloudflare 配置 |
| `tunnel_save_cloudflare_config` | 保存 Cloudflare 配置 |
| `tunnel_save_system_config` | 保存系统配置 |
| `tunnel_start` / `tunnel_stop` | 启动/停止 Tunnel |
| `tunnel_list` | 列出所有 Tunnel |
| `tunnel_create` / `tunnel_delete` | 创建/删除 Tunnel |
| `tunnel_get_config` / `tunnel_update_config` | 获取/更新 Tunnel 配置 |
| `tunnel_start_by_id` / `tunnel_stop_by_id` | 按 ID 启动/停止 Tunnel |
| `tunnel_create_dns_record` | 创建 DNS 记录 |
| `tunnel_register_app_domain` | 为应用注册域名 |
| `tunnel_get_app_domain_status` | 查询域名注册状态 |
| `tunnel_list_credentials` | 列出 API 凭证 |
| `tunnel_create_credential` | 创建 API 凭证 |
| `tunnel_delete_credential` | 删除 API 凭证 |

### 安全建议

- MCP 服务与 API 共用同一套凭证体系，凭证在 API 凭证页面统一管理
- 建议为每个 AI 智能体创建独立的凭证，便于管理和 revocation
- 不要在公开可访问的代码仓库中暴露凭证信息
