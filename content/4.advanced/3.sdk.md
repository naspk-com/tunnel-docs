为了让其他应用方便地调用 Tunnel 管理接口，我们提供了 Python、JavaScript/TypeScript、Go 和 Shell 四种语言的 SDK。所有 SDK 均封装了 API 认证、错误处理和类型定义，开箱即用。

> **MCP 接入**：如果你需要将 Tunnel 管理能力接入 AI 智能体（如 QwenPaw、Claude Desktop），请参考 [MCP 服务文档](/advanced/api-credentials#mcp-服务)，无需使用 SDK。

## Python

### 安装

```bash
pip install fnos-tunnel
```

或从源码安装：

```bash
pip install git+https://github.com/dustink66/fnos-tunnel-sdk-python.git
```

### 快速开始

```python
from fnos_tunnel import TunnelAPIClient

client = TunnelAPIClient(
    "http://<your-fnos-ip>:19092",
    app_id="<your_app_id>",
    app_key="<your_app_key>",
    app_name="com.dustinky.qwenpaw",  # 应用名称，注册域名时使用
    timeout=10.0,
)

# 健康检查（无需认证）
print(client.health())  # True

# 查询 Tunnel 状态
status = client.status()
print(f"Running: {status.running}, Tunnel ID: {status.tunnel_id}")

# 查询域名状态
ds = client.domain_status()  # 不传参数，使用凭证绑定的 appName
print(f"Registered: {ds.registered}")

# 注册域名
result = client.register("qwenpaw.example.com", "http://localhost:19091")
print(f"Success: {result.success}")
```

## JavaScript / TypeScript

### 安装

```bash
npm install fnos-tunnel
```

### 快速开始

```typescript
import { TunnelAPIClient } from "fnos-tunnel-sdk";

const client = new TunnelAPIClient("http://<your-fnos-ip>:19092", {
  appId: "<your_app_id>",
  appKey: "<your_app_key>",
  appName: "com.dustinky.qwenpaw",
  timeout: 10000,
});

// 健康检查
console.log(await client.health()); // true

// 查询 Tunnel 状态
const status = await client.status();
console.log(`Running: ${status.running}, PID: ${status.pid}`);

// 查询域名状态
const ds = await client.domainStatus();
console.log(`Registered: ${ds.registered}`);

// 注册域名
const result = await client.register(
  "qwenpaw.example.com",
  "http://localhost:19091"
);
console.log(`Success: ${result.success}`);
```

## Go

### 安装

```bash
go get github.com/dustink66/fnos-tunnel-sdk-go
```

### 快速开始

```go
package main

import (
    "fmt"
    "time"
    tunnel "github.com/dustink66/fnos-tunnel-sdk-go"
)

func main() {
    client := tunnel.NewAPIClient(
        "http://<your-fnos-ip>:19092",
        "<your_app_id>",
        "<your_app_key>",
        "com.dustinky.qwenpaw",
        10*time.Second,
    )

    // 健康检查
    fmt.Println(client.Health()) // true

    // 查询 Tunnel 状态
    status, _ := client.Status()
    fmt.Printf("Running: %v, PID: %s\n", status.Running, status.PID)

    // 查询域名状态
    ds, _ := client.DomainStatus("")
    fmt.Printf("Registered: %v\n", ds.Registered)

    // 注册域名
    result, _ := client.Register("qwenpaw.example.com", "http://localhost:19091")
    fmt.Printf("Success: %v\n", result.Success)
}
```

## Shell

### 安装

```bash
# 直接下载
curl -O https://raw.githubusercontent.com/dustink66/fnos-tunnel-sdk-shell/main/fnos-tunnel.sh
source fnos-tunnel.sh
```

### 快速开始

```bash
#!/bin/bash
source fnos-tunnel.sh

BASE="http://<your-fnos-ip>:19092"
APP_ID="<your_app_id>"
APP_KEY="<your_app_key>"
APP_NAME="com.dustinky.qwenpaw"

# 健康检查
fnos_tunnel_api_health "$BASE"

# 查询 Tunnel 状态
fnos_tunnel_api_status "$BASE" "$APP_ID" "$APP_KEY"

# 查询域名状态
fnos_tunnel_api_domain_status "$BASE" "$APP_ID" "$APP_KEY" "$APP_NAME"

# 注册域名
fnos_tunnel_api_register "$BASE" "$APP_ID" "$APP_KEY" "$APP_NAME" \
    "qwenpaw.example.com" "http://localhost:19091"
```

### 一键注册

Shell SDK 还提供了自动注册函数，自动完成状态检测和条件注册：

```bash
fnos_tunnel_auto_register "$BASE" "$APP_NAME" \
    "qwenpaw.example.com" "http://localhost:19091"
```

此函数会自动：
1. 检查 Tunnel 状态
2. 查询域名是否已注册
3. 未注册则执行注册，已注册则跳过
